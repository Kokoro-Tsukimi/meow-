<h1 align="center">喵咖魔法书店<br><sub>Meow Café Magic Bookstore</sub></h1>

<p align="center">

## ✨ 特性亮点

- ☕ **咖啡豆积分制** — 独创的虚拟货币系统，用"咖啡豆"管理用户额度，支持兑换码充值和每日签到
- 🔀 **智能渠道调度** — 加权随机选择算法，自动熔断与故障转移，多渠道负载均衡
- 📋 **模型菜单组** — 用户只看到抽象的"菜单名"，真实上游模型标识完全隐藏
- 🛡️ **三档风控引擎** — BLACKLIST 封禁 / SHADOW 影子流量克隆 / DRYRUN 人工审批放行
- 📊 **可视化消费账单** — 逐笔 Token 用量、延迟、缓存命中详情，支持按日筛选和月度导航
- 🎨 **双主题双前端** — 管理端机甲暗色风 / 用户端马卡龙清新风，均支持明暗切换
- 🐳 **Docker 一键启动** — Docker Compose 编排 MySQL / Redis / ClickHouse，双 bat 脚本一键运行
- ☁️ **Cloudflare Tunnel 发布** — 无需公网 IP，一键发布到自定义域名

### 3-1：安装依赖

**小昙**：「在 .env 配置完成之后，就可以保存并关闭了。」

**小昙**：「然后复制以下命令，粘贴到新打开的窗口里并执行。复制之后右键点一下窗口然后按下 Enter 即可。」

```bash
cd gateway
npm install
cd ..\frontend-user
npm install
cd ..\frontend-admin
npm install
```

## 📝 补充说明

**关于数据库初始化**：MySQL 容器首次启动时会自动执行 `gateway/src/db-init.sql`，无需手动运行 `migrations/` 目录下的迁移脚本。`migrations/` 目录保留是为了从旧版本升级的场景。

**技术栈**：全栈 TypeScript — Fastify + MySQL / Redis（Lua 原子计费）/ ClickHouse 后端，双 React + Vite + Tailwind 前端（recharts 图表），Docker Compose 部署，Cloudflare Tunnel 公网发布。
